# Bow for Unity

A functional bow and arrow system featuring a procedural string implementation using Unity's Line Renderer. This package provides an easy-to-use, physics-based archery system for your Unity projects.

</Frame>

## Key Features

<CardGroup cols={2}>
  <Card title="Procedural String" icon="bezier-curve">
    Dynamic bowstring visualization using a Line Renderer that responds to draw strength
  </Card>
  <Card title="Dynamic Force System" icon="gauge-high">
    Projectile strength scales based on draw time, from minimum to maximum force
  </Card>
  <Card title="Physics-Based Flight" icon="arrow-up-right-dots">
    Realistic arrow physics with automatic rotation based on velocity
  </Card>
  <Card title="Easy Setup" icon="wand-magic-sparkles">
    Simple component-based architecture with configurable properties in the Unity Editor
  </Card>
</CardGroup>

## What You'll Get

- **Bow Component**: Handles drawing mechanics, force calculation, and string visualization
- **Arrow Component**: Manages arrow rotation during flight for realistic physics
- **Customizable Parameters**: Fine-tune strength, draw time, and visual feedback
- **First-Person Ready**: Built with FPS camera integration in mind
