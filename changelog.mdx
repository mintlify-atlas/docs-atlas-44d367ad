## Current Version

The Bow Unity project is an actively developed open-source bow and arrow system for Unity.

## Current Features

Here are the features currently implemented in the Bow system:

### Core Mechanics

<CardGroup cols={2}>
  <Card title="Procedural String" icon="wave-pulse">
    Dynamic bow string rendering using Unity's LineRenderer component. The string visually pulls back as you draw the bow.
  </Card>
  
  <Card title="Dynamic Projectile Strength" icon="gauge-high">
    Arrow force scales with draw time. Hold longer for more powerful shots, or quick-fire for rapid but weaker arrows.
  </Card>
  
  <Card title="Easy Setup" icon="puzzle-piece">
    Simple script-based setup with clear Inspector properties. Drag and drop configuration with minimal coding required.
  </Card>
  
  <Card title="Arrow Rotation System" icon="rotate">
    Automatic arrow rotation that aligns with velocity direction for realistic arrow flight physics.
  </Card>
</CardGroup>

### Technical Features

<AccordionGroup>
  <Accordion title="Physics-Based Arrow System">
    - Rigidbody-based physics for realistic projectile motion
    - Force calculation based on camera direction and draw strength
    - Velocity-based rotation for natural arrow orientation
    - Configurable min/max strength values
  </Accordion>
  
  <Accordion title="Visual String Rendering">
    - Three-point line renderer system (upper, middle, lower)
    - Dynamic middle point that moves based on draw percentage
    - Configurable draw back strength for visual feedback
    - Local space rendering for proper bow movement
  </Accordion>
  
  <Accordion title="Customizable Parameters">
    - **minStrength**: Minimum arrow force (default: 2)
    - **maxStrength**: Maximum arrow force (default: 30)
    - **drawTime**: Time to reach full draw (default: 0.35s)
    - **drawBackStrength**: Visual string pull distance (default: 0.05)
    - **arrowRotation**: Default arrow rotation offset
  </Accordion>
  
  <Accordion title="Flexible Architecture">
    - Modular script design (Bow.cs and Arrow.cs)
    - Easy to extend or replace Arrow script with custom behavior
    - Compatible with Unity's prefab system
    - Works with first-person camera setups
  </Accordion>
</AccordionGroup>

## Roadmap

Planned features and improvements for future releases:

### High Priority

<Steps>
  <Step title="Enhanced String Rendering" icon="bezier-curve">
    **Add more dynamic points to the line renderer**
    
    Currently uses 3 points. Planned enhancement:
    - Increase to 5-7 points for smoother curves
    - Implement bezier curve interpolation
    - More realistic string physics and appearance
    
    **Status**: Planned  
    **Difficulty**: Medium  
    **Impact**: Visual quality improvement
  </Step>
  
  <Step title="Code Quality Improvements" icon="code">
    **Clean up the scripts**
    
    Planned improvements:
    - Add XML documentation comments
    - Refactor magic numbers into constants
    - Improve code organization
    - Add input validation
    
    **Status**: Planned  
    **Difficulty**: Low  
    **Impact**: Maintainability and extensibility
  </Step>
</Steps>

### Medium Priority

<AccordionGroup>
  <Accordion title="Easing System" icon="chart-line">
    **Add easing types to the DrawTime**
    
    Implement different easing functions for more natural feel:
    
    | Easing Type | Description | Use Case |
    |-------------|-------------|----------|
    | Sine | Smooth, natural | Default bow |
    | Cubic | Pronounced acceleration | Heavy bow |
    | Quadratic | Balanced curve | Standard bow |
    | Exponential | Dramatic buildup | Powerful bow |
    | Elastic | Slight bounce | Special bows |
    
    **Status**: Planned  
    **Difficulty**: Medium  
    **Impact**: Feel and game juice
  </Accordion>
  
  <Accordion title="Audio System" icon="volume">
    **Add sound effects for drawing and releasing the bow**
    
    Planned audio features:
    - Draw sound (progressive or looping)
    - Release/shoot sound with pitch variation
    - String snap/twang audio
    - Ambient bow creak sounds
    - Volume based on draw strength
    
    **Status**: Planned  
    **Difficulty**: Low-Medium  
    **Impact**: Immersion and feedback
  </Accordion>
</AccordionGroup>
